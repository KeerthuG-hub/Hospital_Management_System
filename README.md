
## Hospital Management System (HMS)

### Project Overview

* A **web-based Hospital Management System** developed to demonstrate real-world **web application design and development**.
* Implements **role-based access control** for Admin, Doctor, and Patient users.
* Includes static informational pages for general users without authentication.

### Technology Focus

* Frontend built using **HTML, CSS, and JavaScript** for structured UI and interactivity.
* Backend handles **server-side logic, authentication, and session management**.
* Database integration for **persistent data storage** and relational data handling.
* Emphasis on **client-side and server-side validation**.

### System Modules

* **Admin Panel**

  * Manage doctors, patients, departments, and appointments.
  * Perform full CRUD operations.
  * Centralized control over system data.

* **Doctor Panel**

  * View and manage appointments.
  * Access patient records.
  * Update diagnosis and treatment details.

* **Patient Panel**

  * User registration and login.
  * Appointment booking and history tracking.
  * View assigned doctor and medical details.
